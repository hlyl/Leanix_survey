# Novo Nordisk Corporate Visual Identity (CVI) Guidelines

> **Source:** Novo Nordisk Corporate Visual Identity PDF (2024) and Corporate Design Manual  
> **Last Updated:** January 2026

---

## 1. Color Palette

### Primary Colors

| Color Name | Hex Code | RGB | Usage |
|------------|----------|-----|-------|
| **True Blue** | `#001965` | (0, 25, 101) | Primary brand color, text, logo |
| **Snow White** | `#FFFFFF` | (255, 255, 255) | Backgrounds, logo on dark |

> **Note:** True Blue and Snow White are the dominant colors. One of them must always be present on any given touchpoint.

### Secondary Colors

| Color Name | Hex Code (Approx.) | Usage |
|------------|-------------------|-------|
| **Light Blue** | `#7AB3E6` | Subtle accentuation, tints |
| **Sea Blue** | `#0055B8` | Accents, guiding users |
| **Ocean Green** | `#4DA398` | Complementary accent |
| **Rose Pink** | `#F5D1D8` | Soft backgrounds, accents |
| **Warm Grey / Sand Grey** | `#C9C0B7` | Neutral backgrounds |
| **Granite Grey** | `#8B8D8F` | Neutral elements |

> **Tints:** Secondary colors can be used in tints from 0-100%. The larger the surface, the lighter the tint. Darker tints are not allowed.

### Spot Colors (Digital UI Only)

| Color Name | Purpose |
|------------|---------|
| **Lava Red** | Error states |
| **Golden Yellow** | Warning states |
| **Forest Green** | Success states |

> **Note:** Spot colors are primarily for digital design to indicate error, warning, or success states. Cannot be used in tints.

### Color Guidelines

**Do's:**
- Use True Blue and Snow White as much as possible
- Use white in any design to keep the overall impression simple and clear
- Use lighter tints (0-100%) of True Blue and secondary palette
- The larger the surface, the lighter the tint

**Don'ts:**
- No gradients or color shadings allowed
- Never apply colored overlays on top of images
- Don't use too many colors at once
- Colors in any transparency must never be mixed or placed on top of each other
- Colors should always be placed on a white background
- Never write text in black—use True Blue for dark text

## 2. Typography

### Primary Font Family

| Font | Usage |
|------|-------|
| **Apis** | Corporate font for all written materials—graphics, illustrations, wordmarks, print, and screen |
| **Apis for Office** | Microsoft Office applications (Word, PowerPoint, Excel) |
| **Arial** | Fallback font (when Apis is unavailable); use for Outlook |
| **Noto Sans** | Non-Latin alphabets only |

### Font Weights

| Weight | Style |
|--------|-------|
| Light | Regular & Italic |
| Regular | Regular & Italic |
| Medium | Regular & Italic |
| Bold | Regular & Italic |
| Extra Bold | Regular & Italic |
| Black | Regular & Italic |

### Font Sizes (Recommended Hierarchy)

| Element | Size (approx.) | Weight |
|---------|----------------|--------|
| **Main Heading / H1** | 48–72pt | Light or Regular |
| **Subheading / H2** | 28–36pt | Light or Medium |
| **Body Text** | 11–14pt | Regular |
| **Small Text / Captions** | 9–10pt | Regular or Light |

> **Note:** Specific sizes vary by touchpoint. Apply a consistent, simple, and clear hierarchy for titles, subtitles, etc.

### Typography Guidelines

**Do's:**
- Always use Apis (or Apis for Office)
- Typography should primarily be in True Blue and Snow White
- Colors from the rest of the palette are allowed—as long as text is readable
- Apply a consistent, simple, and clear hierarchy

**Don'ts:**
- Never pair Apis with other fonts
- Never modify Apis
- Never apply effects such as shadows or glow
- Never write text in black—always use True Blue for dark text
- Do not use condensed or wide font weights of the Apis family

## 3. Logo

### Logo Elements

The Novo Nordisk logo consists of:
1. **The Apis Bull** — A stylized Egyptian Apis bull symbol (heritage since 1920s)
2. **The Wordmark** — "novo nordisk®" in lowercase

### Logo Configuration

| Aspect | Specification |
|--------|---------------|
| **Format** | Always stacked (Apis bull on top of wordmark) |
| **Primary Color** | True Blue (`#001965`) on white or very light backgrounds |
| **Inverted** | White on dark backgrounds, bright colors, or photos |
| **File Formats** | SVG (vector) preferred; PNG for digital |

### Logo Placement

| Placement | Context |
|-----------|---------|
| **Top-right corner** | Presentations, documents, web pages (most common) |
| **Bottom-center** | Single-page materials, posters |
| **Flexible** | Ensure adequate clear space around logo |

### Logo Guidelines

**Do's:**
- Use the True Blue version on white or very light-colored backgrounds
- Use the white version on dark backgrounds, bright colors, or photos
- Maintain adequate clear space around the logo

**Don'ts:**
- Never modify the Apis bull (change shape, color, cartoonify, etc.)
- Never use the Apis bull without the "novo nordisk®" wordmark
- Never use the wordmark without the Apis bull
- Never place the logo on faces, busy backgrounds, or motifs that corrupt integrity
- Never create other logos with graphic symbols—only wordmarks are allowed

## 4. Spacing

### Border Radius

| Element | Radius |
|---------|--------|
| **Buttons** | Rounded (full pill shape recommended) |
| **Cards / Containers** | 4–8px subtle rounding |
| **Input Fields** | 4px |

> **Note:** Novo Nordisk design favors organic, rounded shapes inspired by the circular elements in the logo.

### Padding / Margin Standards

| Context | Base Unit |
|---------|-----------|
| **Desktop** | 16px (1rem) base unit |
| **Mobile** | 12–16px base unit |
| **Icon Canvas** | 100×100px with 4pt stroke |
| **Iconostration Stroke** | 4pt (A4) or 10px (HD canvas) |

### Layout Principles

- **Whitespace is key** — Layouts are characterized by predominant use of whitespace
- **Less is more** — Remove unnecessary clutter and superfluous visual information
- **Simple yet distinct** — Reduce complexity while maintaining visual interest
- **Expression scale** — Ranges from functional (information-heavy) to expressive (impactful/minimal)

## 5. Other Design Guidelines

### Button Styles

| Style | Usage |
|-------|-------|
| **Filled (True Blue)** | Primary actions, CTAs |
| **Filled (White)** | On dark backgrounds |
| **Outline** | Secondary actions |
| **Text Link** | Tertiary / inline actions |

> Buttons should have rounded corners (pill or softly rounded). Avoid sharp corners.

### Icons

| Specification | Value |
|--------------|-------|
| **Canvas Size** | 100×100px |
| **Stroke Weight** | 4pt |
| **Style** | Simple, accessible, convention-following |
| **Colors** | Any from corporate palette |

### Iconostrations

Iconostrations combine icons with flowing strokes for visual storytelling.

| Specification | Value |
|--------------|-------|
| **Stroke Weight** | 4pt (A4) or 10px (HD) |
| **Max Colors** | Up to 3 stroke colors |
| **Fill** | No fill colors allowed |
| **Background** | White or 15% secondary color tint |

### Imagery Guidelines

**Do's:**
- Keep imagery real, authentic, grounded in reportage photography
- Use natural light, saturation, and contrast
- Ensure geographic diversity for identification purposes
- Add photo captions for authenticity (name, condition, country)
- Capture documentary-style snapshots of real life

**Don'ts:**
- No stock photos
- No AI-generated images
- No black & white photography (unless historical)
- No colored filters or overlays on imagery
- Never place text or graphic elements on people's faces

### Photo Captions

| For Patients | For Employees |
|--------------|---------------|
| Name | Name |
| Disease type | Job title |
| Country (optional) | Country (optional) |

### Motion & Animation

| Principle | Description |
|-----------|-------------|
| **Organic in feel** | Ease-in/ease-out, soft effects, subtle finesse |
| **Human touch** | Friendly, trust-building transitions |
| **Informative first** | Guide users, convey status, provide feedback |

## Brand Experience Drivers

All design should reflect the three brand experience drivers:

| Driver | Meaning |
|--------|---------|
| **Authentic** | Be genuine and human—build trust |
| **Simple** | Be straightforward and easy to understand—bring clarity |
| **Distinct** | Stand out and show the way forward—drive change |
